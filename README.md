# Signaling

> A simple signaling server for `y-webrtc`.

This is server is used by [Wildbits](https://wildbits.app).

## **Usage**

```sh
$ signaling --help
Usage: signaling [OPTIONS]

Options:
  -p, --port <PORT>  Server port [env: PORT=] [default: 3000]
  -h, --help         Print help
  -V, --version      Print version
```
